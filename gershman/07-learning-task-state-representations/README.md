# Learning Task-State Representations

**Yael Niv** — *Nature Neuroscience* (2019).
https://www.nature.com/articles/s41593-019-0470-8 · PubMed: https://pubmed.ncbi.nlm.nih.gov/31551597/

**Why it's in the course — claim (a):** "the most difficult part of learning is deciding
what to learn about." RL models of behavior (Rescorla-Wagner, SR) *assume* the state/feature
space; constructing the right representation is the actual problem — and the source of
sample efficiency. Filed under Gershman (Niv is a close collaborator on latent-structure
learning); directly exposes the assumed-representation gap in the RL lectures here.
