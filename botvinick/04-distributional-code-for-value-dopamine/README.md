# A Distributional Code for Value in Dopamine-Based Reinforcement Learning

**Dabney, Kurth-Nelson, Uchida, Starkweather, Hassabis, Munos & Botvinick** — *Nature* 577, 671–675 (2020).
https://www.nature.com/articles/s41586-019-1924-6 · DOI: 10.1038/s41586-019-1924-6

**Why it's in the course:** the modern sequel to the RW→dopamine story. Distributional RL (an AI
advance) predicted that dopamine neurons should encode a *distribution* over future reward, not a
single scalar — and single-unit recordings from mouse VTA confirmed it (diverse optimism/pessimism
across neurons). A clean example of an AI idea generating a confirmed neural prediction — exactly
the "deep RL as a theory of the brain" thesis of this lab, and a rebuttal to "RL models are too
simple to bear fruit."

Lead group: **Matthew Botvinick** (DeepMind), with Uchida (Harvard) for the physiology.
