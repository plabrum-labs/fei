# Reinforcement Learning, Fast and Slow

**Botvinick, Ritter, Wang, Kurth-Nelson, Blundell & Hassabis** — *Trends in Cognitive Sciences* (2019).
https://www.cell.com/trends/cognitive-sciences/fulltext/S1364-6613(19)30061-0

**Why it's in the course — the synthesis / orientation read.** Argues that deep RL is
sample-inefficient ("slow"), that biological intelligence is "fast," and that the bridge is
two mechanisms: episodic memory and **meta-learning**. The most readable single framing of the
sample-efficiency gap this whole study is about — pair it with the structured-priors thesis
(*Building Machines...*) as the two sides of "why is human learning so efficient?"
